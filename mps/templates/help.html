Global Database Tools

  • Global Database Search

    • Use a Global Database Search to search the whole MPS-Db from the Home page. To search:
      • Go to the MPS-Db Home page
      • Type a search term into the Global search bar
      • Type a space and add an additional term(s) (optional)
      • Make preliminary selection (hit enter, click the blue Search button, or mouse down the options list to item of interest and click the left mouse button)
      • Wait for a list of matching information to appear
      • Click to check (or uncheck) filter boxes in the filter section to further restrict (or unrestrict) matching results (optional)
      • Click on provided link to be routed to desired location

  • Table Searching and Exporting

    • In many places throughout the database, there are lists/tables with search bars located at the top. To search:
      • Type a search term in the search bar
      • Type a space and add an additional term(s) (optional)
      • From the list, click on button to View/Edit or click an active link to be routed to desired location

    • Help Notes:
      • Above search bars are options to Copy the table to the clipboard, download the table to CSV, or Print the table
      • The Column visibility button can be used to turn columns on/off in the list/table
      • To sort the list/table, click on the arrow in the column header
      • To sort on multiple columns, click on the arrow on the primary sort column, then hold the shift key and click on additional column arrows
      • Click the blue to see hidden fields

  • Download Data Button

    • There are many locations in the database where a Download data button is provided. To download:
      • Click the Download Data button to download data
      • If shown, check/uncheck box for changing data filtering options (optional)
      • Use options for filtering, if provided (optional)

    • Assay Data Visualization (Intra-Study)

      • The Assay Data Visualization (Intra-Study) features are accessed from within an individual study, pre-filtered as follows:
        • Each treatment group
        • Each plate (if using a plate based MPS Model)
        • Each chip or well (with data table of Assay Data Points)
        • The whole Study

      • Assay Data Visualization - Each Treatment Group:
        • Select a Study from the Study List page
        • Click to View a group
        • Review metadata and Assay Data Point graphs

      • Assay Data Visualization - Each Plate:
        • Select a Study from the Study List page
        • Scroll down to the Plates section and click to View a plate
        • Review metadata and Assay Data Point graphs

      • Assay Data Visualization - Each Chip or Well:
        • Select a Study from the Study List page
        • Scroll down to the Chips and Wells section and click to View a chip/well
        • Review metadata and Assay Data Point graphs
        • Review Assay Data Point table
        • Note: The Assay Data Point table provides the Data Group editor or admin of this Study with the ability to include/exclude Assay Data Points from being included/excluded from graphs and analyses

      • Assay Data Visualization - Whole Study:
        • Select a Study from the Study List page
        • Click the Analyze/Edit button and select Study Summary to be routed to the Study Summary page
        • Review metadata and Assay Data Point graphs for the Assay Data Points

      Assay Data Visualization (Inter-Study)

      • The Assay Data Visualization (Inter-Study) features are available in the Data Analysis tab/icon. Select Graphing/Reproducibility on the Data Analysis icon or tab, then:
        • Check at least one box in each list (work left to right, top to bottom)
          • Notice how checking and unchecking boxes changes the Number of Points (Assay Data Points) selected for inclusion
        • Option 1 - Click on the Show Plots button - to view plots
          • Scroll down the page to see plots data
          • Keep scrolling down the page for group metadata
          • Note: this page works similarly to Assay Data Visualization
        • Option 2 - Click on the Show Reproducibility button - to view plots
          • Use the Custom Filtering for Data Visualization (sidebar) to change grouping or filtering of data
          • Check the Show Details box of the results of interest
          • Scroll down page to detailed results section of interest (including chips, studies and each study's reproducibility status, Max CV, ICC, and graphs
          • Explore the Reproducibility Properties options in the sidebar (optional)
          • Note: this page works similarly to Assay Data Reproducibility

      • To view images and video upload to a study:

        • To perform comparative analysis between media files, downloading the unaltered media file is recommended

    • Assay Data Power Analysis

      A statistical power analysis methodology has been implemented in the MPS-Db to estimate the probability that a statistical test will find a significant difference between two different data groups, such as control and compound treatment groups (post hoc two-sample power analysis), or a significance difference from one treatment group (a priori one-sample power analysis) for an MPS model.

      • To perform power analysis:
        • Click on the Show Details box next to a Target/Analyte (set) of interest
        • Scroll down and select TWO compounds for Power Analysis
        • Scroll down to see additional options (Effect Size Methods and Significance Level)
        • Make desired changes to effective size and significance level selections (optional)
        • Click Run Analysis

      • Help Notes:
        • Customized properties for power analysis are available in the MPS-Db via a sidebar on the power analysis page
          • If hidden, click the Toggle Sidebar button to show the sidebar
          • Make desired selections in the sidebar (for more information on overriding default grouping and filtering, click here )
        • By default, the screen will automatically refresh, to refresh manually, check the Manually Refresh box, make the desired changes, then click on the Refresh button to apply changes

    • Assay Data Reproducibility

      • Customized properties for reproducibility analysis are available in the MPS-Db via a sidebar on the inter-study reproducibility analysis page. To perform reproducibility analysis:
        • If hidden, click the Toggle Sidebar button to show the sidebar
        • Make desired selections in the sidebar
          • Reproducibility Comparison within Selected Data Sets - By Center (Center to Center) or By Study (Study to Study)
          • Normalize by Median - helpful if trends are similar but values are on different scales
          • Maximum Interpolation Size - maximum number of data points that can be interpolated, between experimental measurements, to allow for comparison within selected data sets, center to center or study to study (shown as stars on the reproducibility plots)

      • Help Notes:
        • By default, the screen will automatically refresh, to refresh manually, check the Manually Refresh box, make the desired changes, then click on the Refresh button to apply changes

    • Assay Data Study Sets

      • The Study Sets feature is accessed from the Studies menu. To add a study set:
        • Select Studies
        • Select Add Study Set
        • Select the studies to include in the set
        • Click Continue
        • Fill out the form
        • Click the Select Assays button to limit data to include in the Study Set (optional)
        • Submit

      • Help Notes:
        • A Study Set is conceptually similar to an inter-study analysis, with the added options to:
          • Save the selection of the Study Set for later retrieval
          • Assign the Study Set a URL for easy sharing (e.g., inclusion in a publication)
          • Data from each Study in the Study Set can be limited to a subset of assays
        • Viewers of a Study Set created by another user will only see data from Studies to which they have access (i.e. a Study Set does NOT override existing permissions to Studies)

    • Assay Data Collaborators

      • About Collaborators:
        • A Collaborator Group is a means of allowing data from a private study (a study that has not been signed off) to be shared with another user who is not a member of the same Data Group (e.g., someone from another lab)
        • Using a Collaborator Group is currently the only way to allow someone from another group to see data contained in a Study prior to either signing off a Study or opening the Study to the public
        • Collaborator Groups must be created and edited by an MPS Database Administrator

    • Assay Data Accessors

      • About Accessor Groups:
        • An Accessor Group is a means of allowing data from a study during the year embargo period (after sign off and before public release) to be shared with another user who is not a member of the same Data Group (e.g., someone from another lab)
        • Access Groups must be created and edited by an MPS Database Administrator

    • PBPK Analysis

      • The PBPK Analysis feature is accessed from the Analysis menu
        • Check at least one box in each list (work left to right, top to bottom)
          • Notice how checking and unchecking boxes changes the Number of Points (Assay Data Points) selected for inclusion
        • Click the Show PBPK button
        • Use the Custom Filtering for Data Visualization (sidebar) to change filtering of data (optional)
        • Check a Select box to see details and perform an analysis
        • The primary analyses that can be performed are:
          • Calculate the clearance of a compound
          • Predict the plasma concentration given a dose and dose interval
          • Predict the dose and dose interval to reach a specific plasma level

      • Help Notes:
        • A significant amount of the information used in the PBPK is manually curated into the database
        • Currently, the required parameters and rates have been added to the database for the liver
        • Parameters and rates in yellow can be changed while performing an analysis

    • Disease Portal (Clinical Trials and Study Data Combined via a Disease)

      • To access Disease Portal, select View Diseases from the Models icon or tab, then:
        • Scroll through the list of diseases or use the Search bar to find a Disease of interest
        • Click on the View button next to Disease of interest to see options
          • Disease Overview - disease basics
          • Disease Biology - active links to relevant biology/genomic information
          • Clinical Data - summarized findings from clinicaltrials.gov
          • Disease Models & Studies - list of disease related Models in the MPS-Db and list of studies related to the disease

      • Help Notes:
        • The study list on the Disease Models & Studies page and the summary counts on the main Diseases page are permission specific
        • For more information on the reference data source(s), click here

    • Compound Report (Chemical Properties and Study Data Combined via a Compound)

      • To access the Compound Report feature, select Compound Report from the Compounds icon or tab, then:
        • As a precaution, click the Clear Current Selections button
        • Use the Search bar to limit the compound list
        • Check the boxes for the compounds to include
        • If desired, use the Search bar to get a different compound list
        • If desired, check additional compounds to include
        • Click on the button to Generate Compound Report
        • If a green plus sign is present, click it to see aggregated assay study data

      • Help Notes:
        • Compound selections are cumulative
        • To start with a fresh Search, click the Clear Current Selections button
        • To save a complicated Search, click the Save Selections button (when returning at a later time, click Load Selections)
        • Only normalized data are shown in the aggregated assay study data plots
        • For more information on the reference data source(s), click here

        • Select the Studies icon or tab

    Assay Data Customization Options

      • Customized grouping and filtering is available in the MPS-Db via a sidebar where data are available for review. To customize:
        • If hidden, click the Toggle Sidebar button to show the sidebar
        • Click on the parameter menus (Study Parameters, Cell Parameters, Compound Parameters, Setting Parameters) to see options
        • To filter - click on the filter symbol
          • To select data, check the box next to the data
          • To omit data, uncheck the box next to the data
        • To group - check/uncheck the box in the Group column
          • Check the box to include parameter in grouping
          • Uncheck the box to remove parameter from grouping
        • REPEAT for all attributes to override

      • Help Notes:
        • Changing groups and filters changes the way Matrix Items are grouped into rows in the Group table/list provided below the plots
        • In general, increasing the number of parameters selected for grouping will increase the number of data groups in the summary data aggregation tables
        • By default, the screen will automatically refresh, to refresh manually, check the Manually Refresh box, make the desired changes, then click on the Refresh button to apply changes
      • General graph customization is available via a sidebar where graphs are displayed. To customize:
        • If hidden, click the Toggle Sidebar button to show the sidebar
        • Click on the property menus (Plot Graph By, Data Aggregation, Tooltip Display, Data Display, Time Unit) to see options
        • To change the General Graph Properties, click on the menu of the property of interest
          • Plot Graph By - select By Chips (Matrix Item), By Group, or By Compound to indicate additional aggregation grouping
          • Data Aggregation - select Arithmetic Mean, Geometric Mean, or Median (additional options available based on selection)
          • Tooltip Display - select Each Data Point or Entire Time Point (when hovering over a point on a graph)
          • Data Display - option to select Include Excluded Data and/or treat Negative Values as 0
          • Time Unit - select Day, Hour, or Minute for time unit of X-axis

      • Help Notes:
        • By default, the screen will automatically refresh, to refresh manually, check the Manually Refresh box, make the desired changes, then click on the Refresh button to apply changes
      • Individualized graphing options are available via a graph popup where graphs are displayed. To customize:
        • Right-click on an individual graph (on touch screen - press and hold one finger while tapping with another)
        • Select the desired individualized graph options
          • Show Plot - if uncheck and apply, graph will not be displayed on main page
          • Graph as Percent Control - when box is checked, assays and time points without associated controls are removed and the y-axis units are changed to percent control
          • Graphs as Dose-Response - changes X-axis from time to dose
          • Transform X-axis to Log Scale - transforms X-axis
          • Transform Y-axis to Log Scale - transforms Y-axis
        • Click to View Preview
        • Make changes to the general graph customization features as desired (optional)
        • Click Apply to Plot to apply changes, Revert to Default to remove graph specific customization, or Cancel to go back to the last applied version

    • Adverse Events

        • For more information on the reference data source(s), click here

    • Compound Clustering

      • Select Cluster from the Analysis icon or tab, then:
        • Optional - Click on the buttons to Exclude and/or change data as indicated on the button
          • Notice that options appear, these can be selected from the dropdown
        • Optional - The Minimum Feature Count Per Record field specifies the number of bioactivities that need to be linked to a given option for it to appear
          • e.g. if the min feature count is 10, then the compound "ALDRIN" won't be an option because only 5 bioactivities link to "ALDRIN"
          • Click the "Apply" button to confirm the change to the min feature count
        • Make sure that the boxes are check marked to indicate selections in all tables
        • Click the button to Generate Dendrogram
        • The table on the left shows the bioactivities and drug trial data used in clustering
        • Click on a node to see information about the compound (click the red X button to close)

        • Compound Clustering - Example

          • Follow these steps:
            • Select Target Types - check the box "Single Protein"
            • Select Organisms - check the box "Homo sapiens"
            • Filter Compounds - check all boxes, leave LogP and Molecular Weight blank
            • Select Bioactivities - check the box in the top left to include all
            • Select Targets - check the box "!No Target"
            • Select Compounds - check the box in the top left to include all
            • Select Drug Trials - check the box in the top left to include all
            • Click the button to Generate Dendrogram
      • Help Notes:
        • Not every ChEMBL entry has a corresponding PubChem entry, and vice versa
        • The following will clear all selections
          • Changing between ChEMBL and PubChem; Changing whether questionable results are excluded or not; Changing the min feature count; Changing the Target Types or Organisms
        • For more information on the reference data source(s), click here

    Providing Data

    • Joining a Group

      • Contact an MPS-Db administor to request to add a data group for your team or to be added to an existing team group.

    • Review Study Components

      • Click on the link provided to learn how to review study components (Click to Review Study Components)

      • Help Notes:
        • Study components can be added in various locations in the MPS-Db by clicking on the provided plus signs

    • Add a Study

      • To Add a Study:
        • Click on the Add Study button
          • If the study already exists, click to View All Studies, then click on the click the View/Edit button next to the Study of interest

      • Help Notes:
        • You must be a registered user and be assigned to at least one data group to add a study
        • Everything in the Study can be edited until the Study Data Group Admin signs off the Study (then it is locked)
        • Data contained in files attached as Supporting Data are NOT available for visualization, analysis, or used in calculations within the MPS-GUI
        • To edit a study, go to the study of interest and click on the Anayze/Edit button and select Edit This Study

    • Enter Study Details - Details Tab

      • Add the required information as follows:
        • Group select a Data Group (the team performing the study)
        • Study Type indicate the general purpose of the Study (one is required; check all that apply)
        • Study Name use an informative name (e.g., "Kidney-Liver Vitamin C Metabolism" is more informative than "Study-1")
        • Start Date - start of experiment (typically the day the first cells are added to the device)
        • Description - informative description (purpose, methods, results)
        • Use in Calculations - checking Use in Compound Report interface tells the MPS-Db to use this study's data in the Compound Report Feature
        • Protocol File a Word or pdf file containing the Study protocol
        • Image to Illustrate A Study recommend a picture of the setup or an illustration of the experimental protocol
        • References - add references relating to the this study
        • Click Submit or Next to continue

      • Help Notes:
        • The are many options for what to display when entering the Groups (check and uncheck the Show Cells, Show Compounds, Show Settings, and Show Full Details to explore display options)

    • Make Treatment Groups - Groups Tab

      • Add the required information as follows:
        • Select an MPS Model
        • Select an MPS Model Version (information about cells and settings will be pulled from the model version, if available)
        • Apply to the Group
        • The Type is set based on the device the MPS Model (plate or chip device)
        • Enter the treatment Group Name
        • Enter the Number of Chips in the group
        • Change the Test Type for each group, as needed
        • Add compounds, additional cells, and additional settings by clicking on the Add Compound, Add Cell, or Add Setting button, respectively
        • Click on the Add Group button or click the Copy button (in a Group) to make a new Group
        • Review differences between groups in the Group Difference Display table
        • Click Submit or Next to continue

      • Help Notes:
        • There are many options for the order of adding Groups (e.g., a Group can be Copied at any time as many times as needed)
        • If an MPS Model Version has not been made and/or has not been populated correctly, cells and settings will not be automatically pulled into the group setup
        • An MPS Model Version is a starting point for a treatment group (if changes are made to cells, settings, and/or compounds, a model version will not longer completely be indicative of the conditon of the chip)
        • Treatment compounds will need to be added as they are not part of the MPS model version (compounds cannot be saved as part of a model version)
        • Chips are numbered sequentially, starting from 1 (these can be changed, if necessary)
        • Add cells, settings, and/or compounds by clicking on the Add Cell, Add Setting, or Add Compound buttons, respectively
        • Remove cells, settings, and/or compounds by clicking the red X's (click on individual X's or, to remove a column, click on the red X in the column header, and to remove a row, click on the red X in the row label)
        • Enter a treatment group for each treatment condition (for wells in a plate and for chips)

    • Chips and Plates tabs are displayed based on the Type displayed in the Groups tab.

      Define Chips and Plates - Chips Tab

      • To edit chip names (optional):
        • Click on one of the Rename Chips By buttons
        • Fill out the popup form
        • Click Apply
        • Click Submit or Next to continue
      • To edit chip treatment group association (optional):
        • Click on the dropdown in the Group column
        • Select a different group
        • Click Apply
        • Click Submit or Next to continue

      • Help Notes:
        • Changes can be made to the names and the groups in the same edit session
        • This is all optional

    • Define Chips and Plates - Plates Tab

      • If one or more Type = plate treatment groups were added, wells must be added to a Plate as follows (required):
        • Click the Add Plate button
        • Fill in the Name
        • Select the MPS Model
        • Fill in a Well Prefix (optional)
        • Drag in the plate over the wells to assign a treatment group
        • Fill in the form and click Apply
        • Rename all wells on the plate using an Apply Default button (optional)
        • Click Submit or Next to continue

      • Help Notes:
        • Add as many plates as needed
        • A plate can contain one or more treatment groups
        • A treatment group can span plates
        • Well names must be unique to the study, not to a plate (i.e., if A1 cannot be in more than one plate - use a prefix like P1-A1 and P2-A1)

    • Add Assay Targets and Methods to a Study - Assays Tab

      • Add Assays to a study as follows:
        • Click in the Assay Category box and select a category (optional, but narrows the choices of Targets and Methods if selected)
        • Click in the Target/Analyte box and select a target
        • Click in the Method/Kit box and select a Method
        • Click in the Readout Unit and select the unit to be used for uploading data
        • Click Submit or Next to continue

      • Help Notes:
        • All combinations of Method/Kit, Target/Analyte, and Unit must be listed in the Assays tab PRIOR to uploading assay data

    • Upload Assay Data - Upload Data Tab

      There are multiple ways of uploading data into the MPS-Db, including, 1) processed assay data (non-omic), 2) raw data exported from a plate reader, 3) omic formatted data. These are descrided in more detail below.


      Processed MIF-C-a Format

      • To add processed data in MIF-C-a format:
        • Prepare a data file (MIF-C-a)
        • Make sure that the Target, Method, Unit, and Chip and Well Ids all match, exactly, what was entered as part of the study setup
        • In the Upload Data tab, click the button to add processed data files
        • To get an example template file, click here

      • Processed Data Help Notes:
        • The ONLY assay data that are editable in the MPS-GUI is the Exclude field (editable on the Study Item edit page)
        • Changes made to the Exclude field using the MPS-GUI will be lost when data are re-uploaded
        • To replace assay data values with new values, remove the data file from the Current Data File, then correct the values in the MIF-C-a file and re-upload that file
        • Chip and Well Ids must be unique within a Study
        • Each file (and each workbook if using Excel) should contain data for only one Study
        • Data for one study may be submitted in multiple files (and/or multiple tabs of an Excel workbook)
        • If uploading an Excel workbook, workbook can contain multiple tabs
        • All files (or tabs) with correct headers (i.e. column names) will be evaluated for upload, others will be ignored
        • Correct headers must be placed in the first three rows to be considered valid

    • Validation, Flags and Notes (optional)

      To facilitate data review and validation, the database offers the option to flag and add notes to a 1) Study, 2) Study Item (i.e. a chip or a well), and 3) Plate. When a Study is flagged, a flag is display in the Review column for that Study in the Study List (a note is displayed as a hover tip). When a Study Item is flagged, the flag is displayed in the Review column for that Study Item in the Study Item List. In addition, Study Items can be 'validated' (a check mark will show in the Review column for the Study Item in the Study Item list).


      Study Flags and Notes

      • To add flags and or notes to a Study:
        • Go to the Study of interest
        • Click on the Analyze/Edit button
        • Select Edit This Study
        • Click on the Flag icon at the bottom of the page
        • Click on the Notes button to add a note (adding a note is optional)
        • Click Submit to save

      Study Item Flags and Notes

      • To add flags and or notes to a Study Item:
        • Go to the Study of interest
        • Scroll down to the Chips and Wells section of the page
        • Click to View/Edit a Study Item (chip or well) of interest
        • Click on the Flag icon at the bottom of the page
        • Click on the Notes button to add a note (adding a note is optional)
        • Click Submit to save

      Study Item Validation

      • To add a validation check mark to a Study Item:
        • Go to the Study of interest
        • Scroll down to the Chips and Wells section of the page
        • Click to View/Edit a Study Item (chip or well) of interest
        • Click on the Click Here to Mark this Entry as Validated button near the top of the page
        • Click Submit to save

      • Help Notes:
        • Validation check marks, flags, and notes are visible to all users who have permission to view a study
        • Validation, flags, and notes can be used during the validation process, and removed prior to sharing a study (i.e. study sign-off)
        • Study Item validation is designed to be part of the data provider's QC process (it is optional and is NOT required to be completed prior to study sign off)

    • Data Sharing - Sign-Off a Study (optional)

      Data group administrators can sign-off a Study in their data group.

      • To sign-off a Study:
        • Go to the Study of interest
        • Click on the Sign Off/Permission button
        • Select View/Edit Sign Off Status of this Study
        • Add a note in the Notes box (optional)
        • Click the Click Here to Sign Off on this Study button

      • Help Notes:
        • Study sign off starts a one year embargo period after which the study will be open to the broader scientific community (publicly accessible)
        • Study sign off is optional unless required by project funder
        • During the study sign off process, notes can be added that will be displayed in the study list page when hovering over the check mark in the review column
        • To learn more about options for sharing data with selected colleagues prior to study sign off, see Assay Data Collaborators
        • To learn more about options for sharing data with selected colleagues during the embargo period, see Assay Data Accessors

        • Stakeholder Review

          • Follow these steps:
            • Login to the MPS-Db
            • Go to the study list page - select Studies, then View All Studies or click here
            • Look for studies with a check mark, but without an 'eye' or a 'pencil' in the Review column
            • Click the View/Edit button next to the study to review
            • Click the View/Edit Approval Status of this Study button near the top of the page
            • Add a Note (optional)
            • Click the Click Here to Approve this Study for Release button
            • Click Yes
            • Click Submit
            • Click Overwrite

          • Help Notes:
            • Data providers and/or project funders designate study data stakeholder(s)
            • Designating a study data stakeholder is optional (unless required by the funder)
            • Stakeholders must be registered MPS-Db users to access study data
