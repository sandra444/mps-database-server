Tools

  • Global Database Search

    • Use a Global Database Search to search the whole MPS-Db from the Home page. To search:
      • Go to the MPS-Db Home page
      • Type a search term into the Global search bar
      • Type a space and add an additional term(s) (optional)
      • Make preliminary selection (hit enter, click the blue Search button, or mouse down the options list to item of interest and click the left mouse button)
      • Wait for a list of matching information to appear
      • Click to check (or uncheck) filter boxes in the filter section to further restrict (or unrestrict) matching results (optional)
      • Click on provided link to be routed to desired location

  • Search And Sort Lists/Tables

    • In many places throughout the database, there are lists/tables with search bars located at the top. To search:
      • Type a search term in the search bar
      • Type a space and add an additional term(s) (optional)
      • From the list, click on button to View/Edit or click an active link to be routed to desired location

    • Notes:
      • Above search bars are options to Copy the table to the clipboard, download the table to CSV, or Print the table
      • The Column visibility button can be used to turn columns on/off in the list/table
      • To sort the list/table, click on the arrow in the column header
      • To sort on multiple columns, click on the arrow on the primary sort column, then hold the shift key and click on additional column arrows
      • Click the blue to see hidden fields

    • A statistical power analysis methodology has been implemented in the MPS-Db to estimate the probability that a statistical test will find a significant difference between two different data groups, such as control and compound treatment groups (post hoc two-sample power analysis), or a significance difference from one treatment group (a priori one-sample power analysis) for an MPS model.

      • To perform power analysis
        • Click on the Show Details box next to a Target/Analyte (set) of interest
        • Scroll down and select TWO compounds for Power Analysis
        • Scroll down to see additional options (Effect Size Methods and Significance Level)
        • Make desired changes to effective size and significance level selections (optional)
        • Click Run Analysis

      • Notes
        • Customized properties for power analysis are available in the MPS-Db via a "Sidebar" on the power analysis page
          • If hidden, click the Toggle Sidebar button to show the sidebar
          • Make desired selections in the sidebar (for more information on overriding default grouping and filtering, click here )
        • By default, the screen will automatically refresh, to refresh manually, check the Manually Refresh box, make the desired changes, then click on the Refresh button to apply changes
      • Customized properties for reproducibility analysis are available in the MPS-Db via a "Sidebar" on the inter-study reproducibility analysis page
        • If hidden, click the Toggle Sidebar button to show the sidebar
        • Make desired selections in the sidebar
          • Reproducibility Comparison within Selected Data Sets - By Center (Center to Center) or By Study (Study to Study)
          • Normalize by Median - helpful if trends are similar but values are on different scales
          • Maximum Interpolation Size - maximum number of data points that can be interpolated, between experimental measurements, to allow for comparison within selected data sets, center to center or study to study (shown as stars on the reproducibility plots)

      • Notes
        • By default, the screen will automatically refresh, to refresh manually, check the Manually Refresh box, make the desired changes, then click on the Refresh button to apply changes

    Customizing Queries, Plots, and Analyses (Overriding the Defaults)

      • Customized grouping and filtering is available in the MPS-Db via a "Sidebar" where data are available for review
        • If hidden, click the Toggle Sidebar button to show the sidebar
        • Click on the parameter menus (Study Parameters, Cell Parameters, Compound Parameters, Setting Parameters) to see options
        • To filter - click on the filter symbol
          • To select data, check the box next to the data
          • To omit data, uncheck the box next to the data
        • To group - check/uncheck the box in the Group column
          • Check the box to include parameter in grouping
          • Uncheck the box to remove parameter from grouping
        • REPEAT for all attributes to override
      • Additional graphing options are available on an individual plot
        • Right click on an individual plot
        • Check the desired options
        • Click Apply to Plot

      • Notes
        • Changing groups and filters changes the way Matrix Items are grouped into rows in the Group table/list provided below the plots
        • In general, increasing the number of parameters selected for grouping will increase the number of data groups in the summary data aggregation tables
        • By default, the screen will automatically refresh, to refresh manually, check the Manually Refresh box, make the desired changes, then click on the Refresh button to apply changes
      • General graph customization is available via a "Sidebar" where graphs are displayed
        • If hidden, click the Toggle Sidebar button to show the sidebar
        • Click on the property menus (Plot Graph By, Data Aggregation, Tooltip Display, Data Display, Time Unit) to see options
        • To change the General Graph Properties, click on the menu of the property of interest
          • Plot Graph By - select By Chips (Matrix Item), By Group, or By Compound to indicate additional aggregation grouping
          • Data Aggregation - select Arithmetic Mean, Geometric Mean, or Median (additional options available based on selection)
          • Tooltip Display - select Each Data Point or Entire Time Point (when hovering over a point on a graph)
          • Data Display - option to select Include Excluded Data and/or treat Negative Values as 0
          • Time Unit - select Day, Hour, or Minute for time unit of X-axis

      • Individualized graphing options are available via a graph popup where graphs are displayed
        • Right-click on an individual graph (on touch screen - press and hold one finger while tapping with another)
        • Select the desired individualized graph options
          • Show Plot - if uncheck and apply, graph will not be displayed on main page
          • Graph as Percent Control - when box is checked, assays and time points without associated controls are removed and the y-axis units are changed to percent control
          • Graphs as Dose-Response - changes X-axis from time to dose
          • Transform X-axis to Log Scale - transforms X-axis
          • Transform Y-axis to Log Scale - transforms Y-axis

    Instructions for Data Providers

    {# *anchor* #}
      • Data and metadata collected for inclusion in the MPS-Db are categorized based on when and how they are incorporated into the database. These categories are:
        • Study Context information about the MPS Models, assays, compounds, and cells used in the study (these become options in the study component lists)
        • Study Description the study level details (start date, purpose, methods, and description of the results)
        • Study Design the chips level details (when and what cells, treatments, media, and settings were added to each MPS model)
        • Study Data the resulting measured values, images, and videos

      • It should be noted that there is some overlap between these categories, for example, some study design information will likely be included in the model protocol.

      • The Study Description, Study Design, and Study Data can be added directly to the MPS-Db by a data provider (a registered user who has been granted permission to add study data to the MPS-Db). Adding new components to the study component lists requires special permission that is not typically granted to data providers.
