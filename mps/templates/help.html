Assays

Studies

Adding or Editing a Study

  • Studies serve to organize plates, chips, and their related data
  • The Study Type indicates the general purpose of the study
    Multiple Study Types can be chosen and at least one is required
  • Please try to name the study informatively
    e.g. "Kidney-Liver Vitamin C Metabolism" is more useful than "Study-1"
  • You may not edit the Study ID, for it is automatically generated from your other choices
  • Choosing to Restrict Access will hide the study from anyone who isn't in the listed group
  • You can upload an image, such as a picture of the setup or an illustration of the protocol
    If a preview of the uploaded image does not appear, something went wrong
  • You can upload a PDF or word file which contains the study's protocol
  • A Study Configuration needs to be specified for integrated studies
    If the Study Configuration you need is not available, click the provided link to add one

Study Configurations

Adding or Editing a Study Configuration

  • A Study Configuration shows how different devices in an integrated study associate with eachother
  • Please be sure to use an informative name

Defining How the Models are Associated

  • Each model has a Label which can be one or two characters long
  • Organ specifies which organ model is associated with the label
  • Sequence Number shows the order of input for the models
  • Output lists all of the other models the current model feeds into
    Output should be a list of labels separated by dashes: e.g. "AB-CD" means the model is connected to AB and CD
  • Integration Mode indicates whether or not the model is directly hooked up to the other models
  • Click Add Model to add another row

Assay Layouts

Adding or Editing an Assay Layout

  • An Assay Layout dictates the contents of a plate by showing on a well-by-well basis what compounds and time points were used
  • After selecting a device, a representation of the plate will appear at the bottom of the page
  • If a device you need is not present, contact an administrator
  • You can mark a layout as a "Standard Layout" by clicking the checkbox
    This causes the layout to be at the beginning of lists to make it more noticeable
  • Making a selection from the Action dropdown allows one to manipulate the plate in the following ways:

Set Well Type

  • Selecting this option will allow you to specify the purpose of each well
  • First, choose an option from the dropdown entitled Well Type:
    Then click and drag on the representation so that your selection box contains the wells you want to mark
  • Marking wells with a type can make the layout (and any data associated with it) much more legible

Set Time Point

  • Selecting this option will allow you to specify time points for each well
  • The input Timepoint: is the initial time point and the dropdown next to it specifies the unit
  • You can create a series where the value is increased from well to well
    To do this, set a number in the input Increment:
  • When creating a time point series, you can choose to Add or Multiply
    You can also choose which direction to apply the series
  • Each well can contain only one time point, so previous time points will be overwritten

Add Compound

  • Selecting this option will allow you to specify compounds and their concentrations
  • Wells can contain multiple compounds
  • Similar to time point, you can specify a series for the compound concentration
    The default is Divide, but you can also Add, Multiply, or Subtract
  • Wells can contain only one copy of each compound (e.g. a well can only have one instance of Acetaminophen)

Add Label

  • Selecting this option will allow you to add arbitrary text to the layout's wells
  • This allows users to add data that the other options do not account for
  • After typing into the input Label, click and drag on the desired wells and the text will be added
  • Each well can contain only one label, thus old labels will be overwritten if necessary

Clear Contents

  • Selecting this option will allow you to remove everything in the selected wells
  • Click and drag over the desired wells and all data will be removed
  • Additionally, clicking a time point, compound, or label will let you delete it

Flagging for Review

  • Clicking flags the current layout for review
  • Clicking Notes brings up an input dialog where you can explain why this layout was flagged
  • Clicking removes the flag from the current layout

Chip Setups

Adding or Editing a Chip Setup

  • A Chip Setup describes the protocols taken to prepare a chip
  • After selecting an Organ Model, a link to view its standard protocol will appear (granted a PDF has been uploaded)
  • Variance from Protocol should list the ways this setup diverged from the organ model's standard protocols
  • The Chip ID/Barcode can not be the same as another chip in the study
  • Test Type specifies whether the chip functions as a control or not
    If the chip is not a control chip, you must specify a compound, concentration, and unit

Adding Cells

  • Each row under the header Model Cells describes cells added to the chip
  • If this chip does not have any cells, do not modify this section
  • To get another row, click the Add Cells button
  • Click a magnifying glass to see a list of available cell samples then click the desired row to select that sample
  • Use 'e' for scientific notation: e.g. 1e3 = 1000.
  • If you need a cell sample that is not listed, click the provided link to add it

Flagging for Review

  • Clicking flags the current setup for review
  • Clicking Notes brings up an input dialog where you can explain why this setup was flagged
  • Clicking removes the flag from the current setup

Chip Readouts

Adding or Editing a Chip Readout

  • A Chip Readout describes the assays performed and the raw values from these assays

Uploading Data

  • The format for chip uploads is as follows:

    Chip ID Time Time Unit Assay Object Value Value Unit QC Status
    [Chip ID] [Time] [Time Unit] [Assay] [Object] [Value] [Value Unit] [Leave Blank to Pass]
    [Chip ID] [Time] [Time Unit] [Assay] [Object] [Value] [Value Unit] [Any Value in this Column Marks the Reading Invalid]
  • File uploads can use either full names ("Albumin") or abbreviations ("ALB") for assays
  • Table rows in red will be excluded while rows in green are in the database
  • The brackets ([] and {}) in the above examples and templates are not meant to be taken literally
    When you are actually entering data, remove them
  • The Chip ID should be nothing more than the Chip ID/Barcode you entered in the setup
    i.e. do not include "Chip-" at the beginning of the Chip ID
  • Always start in the top-left corner (the first cell)
  • Uploading overwrites previous data
  • You will have to reselect the desired file after a failed submission

Changing QC

  • QC Status can be changed in the table displayed after data upload
  • To invalidate a reading, type anything into the corresponding QC status input
  • After changing a QC value, click away or hit tab to see the effect on the plots

Flagging for Review

  • Clicking flags the current readout for review
  • Clicking Notes brings up an input dialog where you can explain why this readout was flagged
  • Clicking removes the flag from the current readout

Chip Results

Adding or Editing Chip Test Results

  • Chip Results distill raw data into summaries and important values like half-life
  • First, choose a Readout
    Without a readout, you are unable to specify Assay
  • Summary of Results should be a brief summary of your findings from this chip
    These results can be qualitative
  • Each Result represents a finding from a specific assay
    Clicking Add Result will make another row

Flagging for Review

  • Clicking flags the current results for review
  • Clicking Notes brings up an input dialog where you can explain why these results were flagged
  • Clicking removes the flag from the current results

Plate Readouts

Adding or Editing a Chip Readout

  • A Plate Readout describes the assays performed and the raw values from these assays

Uploading Data

  • You may choose between Block and Tabular formats

  • A theoretical 3x2 6-well plate in Block format would be as follows:

    Plate ID [Plate ID] Assay [Assay 1] Feature [Feature 1] Value Unit [Value Unit 1] Time* [Time 1]* Time Units* [Time Unit 1]*
    [Val 1] [Val 2] [Val 3]
    [Val 4] [Val 5] [Val 6]
    Plate ID [Plate ID] Assay [Assay 2] Feature [Feature 2] Value Unit [Value Unit 2] Time* [Time 2]* Time Units* [Time Unit 2]*
    [Val 7] [Val 8] [Val 9]
    [Val 10] [Val 11] [Val 12]
    *Optional (if not using time, totally exclude cells)

  • The above example in Tabular format would be as follows:

    Plate ID Well ID Assay Feature Value Unit Time* Time Units* Value
    [Plate ID] A1 [Assay 1] [Feature 1] [Value Unit 1] [Time 1]* [Time Unit 1]* [Val 1]
    [Plate ID] A2 [Assay 1] [Feature 1] [Value Unit 1] [Time 1]* [Time Unit 1]* [Val 2]
    [Plate ID] A3 [Assay 1] [Feature 1] [Value Unit 1] [Time 1]* [Time Unit 1]* [Val 3]
    [Plate ID] B1 [Assay 1] [Feature 1] [Value Unit 1] [Time 1]* [Time Unit 1]* [Val 4]
    [Plate ID] B2 [Assay 1] [Feature 1] [Value Unit 1] [Time 1]* [Time Unit 1]* [Val 5]
    [Plate ID] B3 [Assay 1] [Feature 1] [Value Unit 1] [Time 1]* [Time Unit 1]* [Val 6]
    [Plate ID] A1 [Assay 2] [Feature 2] [Value Unit 2] [Time 2]* [Time Unit 2]* [Val 7]
    [Plate ID] A2 [Assay 2] [Feature 2] [Value Unit 2] [Time 2]* [Time Unit 2]* [Val 8]
    [Plate ID] A3 [Assay 2] [Feature 2] [Value Unit 2] [Time 2]* [Time Unit 2]* [Val 9]
    [Plate ID] B1 [Assay 2] [Feature 2] [Value Unit 2] [Time 2]* [Time Unit 2]* [Val 10]
    [Plate ID] B2 [Assay 2] [Feature 2] [Value Unit 2] [Time 2]* [Time Unit 2]* [Val 11]
    [Plate ID] B3 [Assay 2] [Feature 2] [Value Unit 2] [Time 2]* [Time Unit 2]* [Val 12]
    *Optional (if not using time, totally exclude columns)
  • File uploads can use either full names ("Albumin") or abbreviations ("ALB") for assays
  • The brackets ([] and {}) in the above examples and templates are not meant to be taken literally
    When you are actually entering data, remove them
  • The Plate ID should be nothing more than the Plate ID/Barcode you entered in the setup
    i.e. do not include "Plate-" at the beginning of the Plate ID
  • Always start in the top-left corner (the first cell)
  • Uploading overwrites previous data
  • You will have to reselect the desired file after a failed submission

Changing QC

  • In the .csv file you can change a value's QC status by putting a single letter in front of the value
    (e.g. to invalidate the value "12.3" put "x12.3" in the cell instead)
  • Clicking
    will toggle whether or not you can invalidate wells by clicking on the displayed heatmap
  • To invalidate a reading, click the corresponding well (granted the above button is on)
  • Invalidated wells will be greyed out and their values put in red

Flagging for Review

  • Clicking Notes brings up an input dialog where you can explain why this readout was flagged
  • Clicking removes the flag from the current readout
