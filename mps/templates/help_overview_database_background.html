• Database Background

    The Microphysiology Systems Database (MPS-Db) combines the loading, storage, and sharing of experimentally derived assay data and data curated from external reference sources with visualization and analysis tools to provide a powerful platform for designing experimental studies and performing computational modeling. The MPS-Db can store curated assay data and associated metadata from a wide variety of plate-based and microfluidic models. Data from external reference sources are brought into the MPS-Db to facilitate ease of access to those data and to support integration of data across curated data sources.

    Data collected as part of experimental studies are manually curated into the MPS-Db. Data from reference data sources are manually and/or auto-curated. The MPS-Db provides tools to analyze and/or visualize data pulled from external data sources, and also provides tools to analyze curated study data in combination with reference data.
